Инструкция по установке
Windows:
	1. Установите python
	2. Откройте консоль
	3. Перейдите в каталог с app.py 
	4. Пропишите install_requirements.bat
	5. Перейдите в database и выполните настройку логинов и паролей ( см инструкция по логинам )
	6. Выйдите из каталога database и пропишите python app.py
Linux:
	1. Установите python
	2. Откройте консоль
	3. Перейдите в каталог с app.py 
	4. Пропишите install_requirements.sh
	5. Перейдите в database и выполните настройку логинов и паролей ( см инструкция по логинам )
	6. Выйдите из каталога database и пропишите python3 app.py
ИНСТРУКЦИЯ ПО ЛОГИНАМ
	Добавление логина: Откройте logins.txt и для каждого нового логина пишите текст по формату с новой строки:
	<ЛОГИН>:<ДОСТУП>
	Где доступ: a - Администратор u - Пользователь
	Добавление пароля: Откройте passwords.txt и для каждого нового пароля пишите текст с новой строки:
	Пример установки логинов и паролей:
		logins.txt:
			astra:a
			usurus:u
		passwords.txt:
			astra56
			usurus56
	Будет создан логин:
		Администратор:
			Логин: astra
			Пароль: astra56
		Пользователь:
			Логин: usurus
			Пароль: usurus56
